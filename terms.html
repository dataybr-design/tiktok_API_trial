Términos de servicio

Esta aplicación es una prueba técnica interna destinada a validar la conexión con la API oficial de TikTok.

Solo deben autorizar la aplicación usuarios que tengan permiso para conectar la cuenta TikTok correspondiente.

La aplicación se utilizará exclusivamente para consultar métricas y datos autorizados por TikTok y por el titular de la cuenta.

El uso de esta aplicación está sujeto a las políticas y condiciones de TikTok.
