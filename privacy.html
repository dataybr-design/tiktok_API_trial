Política de privacidad

Esta aplicación es una prueba técnica interna para conectar cuentas TikTok autorizadas y consultar datos de rendimiento mediante la API oficial de TikTok.

La aplicación solo accede a los datos autorizados por el titular de la cuenta durante el proceso de autorización. Los datos se utilizan únicamente para análisis interno y reporting.

No se venden datos personales a terceros. Los tokens de acceso se almacenan de forma privada y no se exponen en el navegador.

Para solicitar la eliminación de datos asociados a esta prueba, contacta con el equipo responsable del proyecto.
